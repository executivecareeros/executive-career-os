# Executive Language

> **Purpose:** Define language that respects executive experience, preserves agency, and supports clear decisions.

## Executive-First Language

An executive is not merely an applicant moving through a funnel. Write about the whole career, the decision being made, and the relationships involved.

## Words We Prefer

| Prefer | Why |
|---|---|
| executive | Recognizes the person and career context |
| opportunity | Allows evaluation beyond a posted job |
| career decision | Emphasizes agency and consequence |
| objective | Connects action to stated direction |
| principle | Identifies a governing commitment |
| constraint | States a boundary without judgment |
| trade-off | Acknowledges competing value |
| evidence | Grounds a claim |
| assessment | Describes structured evaluation |
| recommendation | Preserves the executive’s final authority |
| confidence | Describes evidence quality and completeness |
| missing information | States a knowledge gap directly |
| correction | Preserves original history while repairing it |
| career history | Recognizes enduring context |
| relationship | Reflects the human connection beyond a contact record |
| compensation | Includes the full package and terms |
| leadership scope | Goes beyond title alone |
| career capital | Describes accumulated future choice without reducing it to a score |

## Words We Avoid

| Avoid | Use instead |
|---|---|
| candidate, applicant | executive, except in an employer-process context |
| dream job | target opportunity, aligned opportunity |
| perfect fit | strong alignment with stated conditions |
| guaranteed | supported, confirmed, or uncertain as evidence permits |
| success probability | confidence in the available evidence |
| rejection failure | application outcome or decision outcome |
| weakness | development area, missing evidence, or unmet requirement |
| job hunting | opportunity discovery or executive search |
| hustle | sustained action, relationship development, preparation |
| hack | method, practice, or process |
| game-changing | name the specific consequence |
| effortless | describe the actual work removed |
| unlock | enable, provide, or clarify |
| power user | experienced executive or advanced use, where needed |
| our data | the executive’s data or product data, as applicable |

## Tone Rules

- Respect experience without assuming infallibility.
- Be direct without issuing commands.
- Name difficult outcomes without euphemism or blame.
- Do not equate seniority with status or personal worth.
- Do not flatter titles, compensation, employers, or ambition.
- Avoid recruitment clichés and motivational language.
- Recognize personal and family constraints as legitimate decision context, never as lack of commitment.

## Decision Language

Preferred structure:

- **Recommendation:** Review carefully.
- **Evidence:** The role expands regional scope and matches the stated industry preference.
- **Conflict:** Travel exceeds the confirmed maximum.
- **Missing:** Total compensation and relocation support.
- **Next decision:** Confirm whether the travel requirement is negotiable.

This structure should remain recognizable across interfaces, reports, conversations, and briefs.
