# Import Privacy
> Purpose: State consent, retention, and handling requirements for professional-history sources.

Consent records source, sanitized filename, Workspace, Executive Identity, version, timestamp, and retention choice. Raw content is neither logged nor persisted before review. Files default to discard after extraction. “Retain reference” records intent only until encrypted private document storage, malware scanning, quotas, lifecycle deletion, and access auditing exist. Nothing is shared automatically.
