# Professional History Import
> Purpose: Define the deterministic, review-first path from external history to Career Memory.

Plain text, CSV, structured JSON, and manual entry are parsed. LinkedIn is supported only as an exported CSV; there is no connection or scraping. PDF and DOCX are preview-only because no reliable parser is installed. Career Passport is an architecture placeholder.

Extraction creates a temporary review draft. Unknown values remain unknown. The executive accepts, edits, rejects, merges, or marks items uncertain before authenticated repositories persist them. Deterministic checks identify date order, overlaps, duplicates, gaps, current-role inconsistencies, and related review items without automatic resolution.

Confirmed roles produce meaningful append-only Ledger events. Historical employment dates are `occurredAt`; import time is `recordedAt`. Blueprint outputs are suggestions requiring approval. The first Executive Brief separates facts, suggestions, missing information, conflicts, and actions.
