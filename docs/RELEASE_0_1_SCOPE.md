# Release 0.1 Scope
> **Purpose:** Define what the audited release demonstrably includes and excludes.
## Included
Executive dashboard and shell; design system; opportunity, company, application, archive, compensation, discovery, Blueprint, Knowledge Network, Atlas decision/agent demonstrations; product entitlements; local fictional datasets; strategic and architecture documentation.
## Demonstration Guarantees
Routes compile, shared navigation is present, domain contracts are strongly typed, history is modeled append-first, Atlas reasoning is deterministic, premium states are transparent, and demonstration status is stated in primary data-driven workspaces.
## Excluded
Authentication, authorization, persistence, multi-tenancy, billing, AI/model providers, live discovery, APIs, scraping, browser automation, outbound notifications, document processing, voice, and production security controls.
## Release Position
Release 0.1 is an architecture and product demonstration. It is not ready to hold real executive data or execute autonomous actions.
