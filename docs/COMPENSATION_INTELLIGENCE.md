# Compensation Intelligence

Every range, estimate, target, minimum, request, budget, offer, counteroffer, revision, and outcome is a separate immutable record linked by existing IDs. Future compensation changes must also append a Career Ledger event.

Formatting preserves original currency and distinguishes disclosed, estimated, candidate, and undisclosed values. Comparisons require compatible currencies; no live rates or cross-currency aggregation exists. This is not financial or tax advice. All current values are fictional demonstrations.
