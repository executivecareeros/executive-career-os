# Interaction Guidelines
> Purpose: Standardize predictable behavior across Atlas workspaces.

- Primary actions advance the current executive task; secondary actions navigate or defer.
- Focus Mode removes persistent navigation while preserving the desktop command bar.
- Today, Priorities, and Reasoning remain one click from every desktop page.
- Empty states explain why the area is empty and provide one useful next action.
- Filters update visible results without changing stored records.
- Destructive or permanent actions require explicit confirmation.
- Loading, errors, and unknown values must be stated rather than inferred.
