# Atlas Design Language
> Purpose: Define the visual character shared by every Executive Career OS surface.

Atlas uses restrained dark surfaces, cool neutral typography, blue for orientation, emerald for confirmed readiness, amber for review, and rose only for material risk. Information density is executive: concise headings, factual supporting copy, visible provenance, and progressive disclosure.

The hierarchy is: page purpose, current decision state, priority information, supporting evidence, then secondary controls. Cards group one coherent decision object. Metrics use compact labels and never compete with the page title.
