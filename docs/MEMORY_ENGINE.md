# Memory Engine
> Purpose: Document deterministic memory derivation and explanation rules.

Rules require minimum confirmed evidence counts. Repeated travel-related declines, PE-backed decisions, compensation gaps, and board roles can produce observations. Unconfirmed inputs are ignored. Contradictions reduce confidence. Thresholds are Low, Moderate, High, and Very High. Every output lists supporting, contradicting, and missing evidence.

The engine contains no AI provider, LLM, prediction, generated prose, chat, or external connection. Future providers may consume snapshots but cannot bypass evidence and confirmation rules.
