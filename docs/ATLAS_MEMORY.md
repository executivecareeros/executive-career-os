# Atlas Memory
> Purpose: Define the transparent meaning layer built above confirmed career history.

Career Ledger answers what happened. Atlas Memory records what deterministic rules have observed across confirmed records. Every memory contains evidence, source references, reason codes, confidence, missing evidence, contradictions, timestamp, lifecycle, and version.

Atlas uses observational language: “I observed…” or “Based on previous confirmed decisions…”. It does not silently convert patterns into preferences. Sources are limited to confirmed Blueprint, Ledger, application, company, compensation, knowledge, discovery, import, and Executive Brief records.

Lifecycle states are Active, Confirmed, Questionable, Superseded, Archived, and Forgotten. State changes create append-only audits; records never disappear silently.
