# ODS Templates

> Purpose: Provide minimum reusable structures for consistent work without creating parallel authoritative registers.

Use a template inside the existing authoritative location for the work. Do not create a new register merely because a template exists.

| Template | Use |
| --- | --- |
| [Work Order](WORK_ORDER_TEMPLATE.md) | Scope, authority, plan, and gates before execution |
| [Decision Record](DECISION_RECORD_TEMPLATE.md) | Material product, technical, security, or company decision |
| [Release Record](RELEASE_RECORD_TEMPLATE.md) | Release evidence, deployment, acceptance, and rollback |
| [Completion Report](COMPLETION_REPORT_TEMPLATE.md) | Final factual handoff and closure |
| [Protocol Exception](PROTOCOL_EXCEPTION_TEMPLATE.md) | Temporary, approved deviation from ODS |
