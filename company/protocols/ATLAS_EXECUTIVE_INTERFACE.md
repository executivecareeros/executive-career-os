# Atlas Executive Interface

> Atlas is the Executive Intelligence Interface for Orendalis, not a generic chatbot. Atlas recommends; executives decide.

## Three connected experiences

### Personal Atlas

Private assistance for career strategy, opportunities, companies, compensation, interviews, coaching, CV/documents, board/advisory work, workspace operations and personal decision history.

### Atlas Everywhere

A persistent, context-aware panel or command interface across company, opportunity, dashboard and workspace surfaces. It receives the current canonical object and permitted executive context without inventing state or crossing workspaces.

### Atlas in Executive Rooms

Atlas participates only when explicitly mentioned, invoked through an approved command, or included in a moderator-authorized scheduled summary. Basic messaging never depends on AI.

## Response contract

Every material response separates:

1. Explicit user preference.
2. Verified fact and source.
3. Atlas interpretation.
4. Unknown/unavailable/conflicted information.

Recommendations include summary, evidence, confidence, reasons for/against, alternatives, unknowns and next actions. Atlas must be able to answer “I know this because…”. It never conceals uncertainty, changes a decision automatically or rewrites immutable reasoning history.

## Context and privacy

- Workspace context is least-privilege and purpose-bound.
- Page context uses canonical IDs, not copied unverified prose.
- Room context respects room visibility, membership, retention and moderator controls.
- Private Atlas history is not exposed to Rooms.
- Preferences persist only after explicit executive action and remain distinguishable from verified facts.

## Deterministic-first execution

Use stored intelligence, cache, fingerprints, deterministic extraction and accepted analysis before any AI call. AI is permitted only when measurable decision value exceeds deterministic alternatives. Accepted AI outputs are versioned, cached where lawful, attributed and invalidated when source fingerprints change.

## Current evidence

Structured opportunity review, decision workspace, evidence visibility, unknown handling, preference preservation and immutable snapshots exist and pass deterministic contracts. Persistent Atlas Everywhere, company/opportunity comparison UX, Room commands and live confidence calibration are incomplete.

## Acceptance criteria

- Available on company, opportunity, dashboard and workspace without navigation loss.
- Correct canonical context and workspace isolation proven.
- Every factual statement links to evidence/confidence/freshness.
- Unknowns remain explicit; unsupported recommendation is withheld.
- User preferences persist and can be inspected/changed.
- No automatic executive action.
- Desktop, 390px, keyboard and screen-reader acceptance.
- Live usefulness, interruption and decision-quality metrics established.
