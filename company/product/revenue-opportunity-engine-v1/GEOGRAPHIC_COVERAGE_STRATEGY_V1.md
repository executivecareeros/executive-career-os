# Geographic Coverage Strategy v1

- **Version:** `tmci-geography-v1`
- **Status:** Active network strategy
- **Effective date:** 2026-07-17
- **Scope:** Opportunity acquisition and coverage measurement

## Separation of Concerns

The Opportunity Network determines which opportunities exist. It optimizes for market opportunity, lawful access, source reliability, freshness, quality, and strategic growth potential.

An Executive Profile determines which opportunities an individual sees and how they are ranked. Citizenship, work authorization, location, relocation, language, and preferences never alter network acquisition targets or the Target Market Coverage Index (TMCI).

## Global Core Markets

Coverage is measured independently for:

1. G20 economies
2. Worldwide Remote
3. European Union
4. EMEA
5. United Kingdom
6. North America
7. MENA
8. APAC
9. Latin America
10. Africa

Expansion is demand-led rather than uniform. Within every market, priority goes to internationally hiring employers that publish structured, reliable opportunity data through supported ATS platforms and recruit executive commercial leadership across countries.

## Remote Classification

Remote opportunities retain an evidence-backed scope:

- Worldwide Remote
- EMEA Remote
- EU Remote
- North America Remote
- APAC Remote
- LATAM Remote
- MENA Remote
- Country-restricted Remote
- Timezone-restricted Remote
- Unknown Remote

The word “Remote” alone is not evidence of worldwide eligibility. Missing scope remains `unknown`.

## TMCI Contract

Every region is scored from 0–100 using factual network measurements:

| Metric | Weight |
| --- | ---: |
| Employer coverage | 15% |
| ATS coverage | 10% |
| Executive-title coverage | 15% |
| Industry coverage | 10% |
| Freshness | 12% |
| Canonical quality | 15% |
| Executive opportunity density | 10% |
| Remote coverage | 5% |
| Work-authorization clarity | 5% |
| Compensation completeness | 3% |

Regional weights are versioned in the executable contract. A missing region contributes zero; absence is never excluded from the denominator. TMCI inputs contain network measurements only and do not accept Founder or executive profile data.

## Operating Measures

For every region, operations records:

- employers and supported ATS platforms;
- industries and executive titles;
- active, fresh, canonical, and remote opportunities;
- duplicate rate and canonical quality;
- work-authorization clarity;
- compensation completeness;
- opportunity density relative to the target employer cohort.

Each coverage cycle expands the largest measured gap with the best combination of executive value, lawful accessibility, reliability, and engineering efficiency. Provider count is not a success metric.

## Change Control

Changes to region definitions, weights, or metric definitions require a new strategy version. Historical TMCI results retain their original version so changes cannot rewrite prior performance.
