# Executive Review Flow

Last updated: 2026-07-17 · Owner: Atlas Product · Version: 1

`Canonical opportunity → bounded Knowledge Graph view → M6 Opportunity Intelligence assessment → M9 experience objects → Atlas Opportunity Review → executive investigation or decision → M7 validation`

## Eligible path

All five gates pass → Recommendation Available → evidence and confidence remain visible → executive reviews reasons for and against → next action → decision summary.

## Withheld path

Any gate fails → Recommendation Withheld → no Recommendation object → missing evidence, Unknowns, conflicts, and alternatives remain visible → minimum investigation request → reassess only after new reviewed evidence.

## Safety

The flow never changes the graph, assessment, recommendation history, or executive decision. Atlas orchestrates existing evidence and semantic objects; the executive retains final authority.
