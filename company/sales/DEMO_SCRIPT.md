# Demo Script

> Purpose: Demonstrate the product as one coherent executive operating system.

1. Start with Today’s Brief and current priorities.
2. Show Career Ledger as permanent history.
3. Show Executive Blueprint as confirmed direction.
4. Review an opportunity with evidence and trade-offs.
5. Open Atlas Memory and distinguish known, suspected, and unknown.
6. Demonstrate reasoning alternatives and confidence questions.
7. Prepare an interview or negotiation.
8. Close with data ownership, portability, and executive control.

Use fictional demonstration data and state limitations plainly.
